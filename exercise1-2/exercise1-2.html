Exercise 1.2

Your sketch:

Instructions

Open exercise1-2/sketch.js. It contains the code from the slidingCircleV3 example shown in lecture. Run the code to remind yourself of what it does.

Your task is to make one tiny edit to the code so that the circle slides back and forth at a random speed between 1 and 10.

To achieve the expected output, you will need to do the following:

  • Identify the variable that determines the speed of the circle.
  • Change the initial value of that variable from a hardcoded number to the result of a call to random().
  • Initialise the variable in the appropriate part of the code (see the randomColours example from last lecture for a clue).

When you are done, check your work using check-exercise1-2.html.

Expected output:

You only need to edit sketch.js.

If you have edited the variable to use random() but the circle is not moving, it is likely that you have called random() in the wrong place. Although you can initialise a variable with a hardcoded value outside of setup() and draw(), you can only call p5.js functions from inside setup() or draw(). You will need to declare the variable above setup(), then initialise it by calling random() in setup(). The randomColours example from last lecture includes some very similar code.

Make sure you are calling random() with two arguments--the minimum and maximum possible values.
