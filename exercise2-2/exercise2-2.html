Exercise 2.2

Your sketch:

Instructions

Open exercise2-2/sketch.js, add the setup() and draw() functions, and create a 400 by 400 pixel canvas.

Create a 50 pixel wide square that moves with the mouse. You will notice that when the mouse moves out of bounds of the canvas, the square disappears. Use conditionals to keep the square fully visible on the canvas. When the mouse leaves the canvas, the square should stick to the nearest edge (see the video below). It is recommended that you tackle this problem one edge of the canvas at a time e.g. start by figuring out how to keep the square on the canvas if mouseX is too far to the left.

Optional extension: When you have successfully completed this exercise using conditionals, you may like to try using p5.js's constrain() function. It makes the code much simpler!

When you are done, check your work using check-exercise2-2.html.

Expected output:

You only need to edit sketch.js.

Start by drawing a square that moves with the mouse. It is recommended that you draw the square in CENTER mode. You should remember how to do this from the last practical. If not, revisit the sample solutions for the last practical.

Next, try to keep just the x coordinate within bounds of the canvas. Here are the key considerations (assuming the square is drawn in CENTER mode):

  • The square will start to disappear off the left edge when mouseX is less than half the width of the square. To keep the square in bounds on the left, set its x coordinate to the lowest "safe" value of x if mouseX would otherwise cause it to move off screen.
  • The square will start to disappear off the right edge when mouseX is greater than the width of the canvas minus half the width of the square. To keep the square in bounds on the right, set its x coordinate to the highest "safe" value of x if mouseX would otherwise cause it to move off screen.

Finally, you will need to write code to keep the square in bounds on the y axis. This should be very similar to the code you wrote for the x axis, you're just dealing with different variables.
