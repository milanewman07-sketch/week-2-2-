Your sketch:

Instructions

Open exercise1-1/sketch.js. It contains the code from the mousePressAppear example shown in lecture. Run the code to remind yourself of what it does. Try clicking your mouse to make the circle appear and disappear.

Your task is to edit the sketch to create the output shown in the video below. When the mouse is pressed, a new circle should appear on screen at a random location for every frame that the mouse is pressed. The circles should also stay on screen rather than disappear. Make the following changes:

  1. Change the arguments passed to circle() be randomly generated. The x coordinate should be a number between 0 and the width of the screen. The y coordinate should be between 0 and the height of the screen. The diameter of the circle should be between 10 and 100 (inclusive).
  2. If you run the code now, you'll see that circles rapidly appear and disappear when the mouse is pressed. This is because background() is called every frame. Remember that the background() function doesn't work the way you might expect if you've used tools like Photoshop, which allow you to paint a layer behind existing shapes. Instead, background() fills the screen with colour, painting over any existing shapes. This causes the flashing effect, where a circle is drawn then immediately painted over on the next frame. To allow the circles to stay on screen, move the background() call into setup(), after the canvas is created.
  3. Next, see if you can work out how to clear the canvas when a key is pressed. You will need to use the keyIsPressed system variable and an if statement. What goes inside the if statement? The clue is in the previous bullet point! The video below shows the screen being cleared at 7 seconds.

When you are done, check your work using check-exercise1-1.html.

Expected output:

You only need to edit sketch.js.

For step 1, replace the existing arguments passed to circle() with calls to the random() function. For the x and y locations, you'll only need to pass random() one argument--the largest number that it can generate. For the circle width, you'll need to pass random() two numbers: the first should be the smallest number it can generate and the second should indicate the largest number it can generate.

For step 2, delete the call to background() that is currently in draw() and put it at the end of setup() instead.

For step 3, the syntax of the if statement should be very similar to the if statement that is already in the sketch but you will need to use the keyIsPressed system variable instead of mouseIsPressed. To clear the screen, when a key is pressed, call background() inside the if block. This will paint over all the circles that have been drawn.
