Exercise 2.1

Your sketch:

Instructions

Open exercise2-1/sketch.js, add the setup() and draw() functions, and create the canvas at a size of your choosing.

Choose a background colour and a fill colour and draw a shape of any size (but smaller than the canvas) anywhere on the canvas. When the mouse button is pressed, the background colour and fill colour should swap. You can see an example below. When the mouse is not pressed, the background is black and the fill is pink. When the mouse is pressed, the background is pink and the fill is black.

You will need to use the mouseIsPressed system variable and write a conditional statement. The mousePressAppear example from last lecture may help you to get started.

When you are done, check your work using check-exercise2-1.html.

Expected output:

You only need to edit sketch.js.

It is recommended that your conditional statement has an else branch. You should also use the mouseIsPressed system variable in the boolean expression. As mouseIsPressed always has a boolean value, it can be used as a boolean expression.
